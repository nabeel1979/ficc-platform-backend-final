# FICC Scanner Client - دليل التثبيت

## المتطلبات
- Windows 10/11
- Python 3.8+ — تحميل من https://python.org
- سكنر متوافق مع WIA

## التثبيت
1. ثبّت Python من https://python.org (✅ فعّل "Add to PATH")
2. شغّل `install.bat` كمدير
3. شغّل `run.bat`

## طريقة الاستخدام
1. افتح صفحة المراسلات على https://ficc.iq/correspondence
2. أنشئ كتاباً جديداً أو افتح مسودة
3. **انسخ رقم المسودة** (يظهر في الرسالة بعد الحفظ)
4. افتح برنامج FICC Scanner
5. الصق رقم المسودة + رمز الدخول
6. اضغط "مسح صفحة واحدة" أو "فيدر"
7. اختر السكنر → يُرفع تلقائياً للنظام ✅
